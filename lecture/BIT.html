小技巧

利用位元運算可知道

x&(-x)會得到x+該數值可到達最近的維護位置

#define lowbit(x) (x&(-x))

建立/修改

觀察後可知道建立第c位置時

等價於在原本更新到c-1的bit中增加位置c的值

因此從位置c開始一路修改到邊界即可

void add(int c,int val){
    for(;c<=N;c+=lowbit(c))
        bit[c]+=val;
}

查詢

從點c一路往前查到2的冪次即可

int qry(int c){
    int ret=0;
    for(;c;c-=lowbit(c))
        ret+=bit[c];
    return ret;
}

逆序數對

給你一個數列

請找出有多少組 ai>aja_i>a_j且i<ji<j

也就是說 求有多少組前面的比後面的數字小

前情提要-離散化

給你一些數字

請按照順序輸出他們的是第幾小的數字

作法

  1. 複製原本的陣列成陣列tmp
  2. 先將tmp中重複的數字刪除
  3. 將tmp按照大小排列
  4. 可以知道第N大的數字會出現在tmp的第N項
  5. 查詢原本的陣列每一項在tmp中的順序

CODE

#define all(a) a.begin(),a.end()
// vec是要排列陣列
vector<int> tmp=vec;
tmp.resize(tmp.unique(all(tmp))-tmp.begin);
sort(tmp.begin(),tmp.end());
rep(i,sz(vec))
    vec[i]=lower_bound(all(tmp),vec[i])-tmp.begin();

逆序數對想法

  1. 先把所有數字離散化
  2. 從數列後往前掃過
  3. 同時開一個陣列K 紀錄離散化後的數字是否出現
  4. 遇到每一個數字只需要查看在K中 該數字的位置之前的合數字
  5. 可以BIT查詢數字!
